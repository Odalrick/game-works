## UX Enhancements

- [ ] **U1: Right-click tile to cycle feedback backwards** — Currently click cycles white -> orange -> green -> white. Right-click should go green -> orange -> white -> green.
- [ ] **U2: Prefill known green letters in new guess** — When adding a guess, pre-populate tile feedback for letters already known to be green in that position.
- [ ] **U3: Replace "endsWith" quest rule with "lettersAt"** — Five textboxes, one per position. Fill in any subset to require those letters at those positions. Subsumes endsWith (and the missing startsWith). "use" and "avoid" remain as-is.
- [ ] **U3a: Select-all on quest letter input focus** — When clicking the letter input, select all text so typing a letter overwrites immediately.
- [ ] **U4: Highlight conflicting green letters** — Red border on tiles where: (a) two greens in the same column disagree, or (b) a column has a green letter and the same letter appears non-green in that column. Visual feedback for data-entry mistakes.
